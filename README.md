### 游戏特色 🔥

- 纯前端实现，0 服务器，秒开秒玩
- 超美毛玻璃 + 渐变背景 + 动画特效
- 两种模式：
  1. **普通模式**：10 次机会，硬核挑战
  2. **作弊模式**：无限次 + 一键看答案（认输专用）
- 支持手机、平板、电脑全屏适配
- 回车提交，操作丝滑

---

### 游戏规则

1. 系统随机生成一个 4 位数字（允许前导零，如 `0071`）
2. 你输入 4 位数字猜测
3. 反馈说明：
   - `猜中了` → 恭喜通关！
   - `高了` / `低了` → 数字大小提示
   - `位对码错，高了` → 有数字对了但位置错了 + 大小提示
