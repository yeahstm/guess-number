### 游戏特色

- 两种模式：
  1. **普通模式**：10 次机会，硬核挑战
  2. **作弊模式**：无限次 + 一键看答案（认输专用）

---

### 游戏规则

系统随机生成一个 4 位数字（允许前导零，如 `0071`）
高了:猜测的4位数字整体大于密码
低了:猜测的4位数字整体小于密码
位对码错，高了/低了:高低判断同上，且存在数字与密码一致但位置错误的情况(数量不明确)

其中值得注意的是，位对码错的判断，是猜测的数字中有与目标相同的数字，但位置不一致，且该位置上的数字猜测的是错误的才判断为位对码错，
如：
目标为1234
猜测数为1243，显示位对码错；但如果猜测数为1233，虽然第四位3在目标数第三位中有，但是因为猜测数第3位已经为3，故不判断为位对码错
